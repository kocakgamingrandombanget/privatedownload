Installation GuidePanduan Pemasangan

1

Download FileUnduh File

Open the My Add-ons tab and click the Download button. Wait for the file to finish downloading to your device. Buka tab Add-on Saya dan klik tombol Unduh. Tunggu hingga file selesai diunduh ke perangkat Anda.

2

Install to MinecraftPasang ke Minecraft

If you downloaded the Realms version, simply double-click it. If you downloaded the BDS version for servers, extract it into your `behavior_packs` and `resource_packs` folders respectively. Jika Anda mengunduh versi Realms, cukup klik ganda file tersebut. Jika versi BDS untuk server, ekstrak ke dalam folder `behavior_packs` dan `resource_packs`.

3

Apply to WorldTerapkan ke Dunia

Go to your World Edit menu. Make sure to apply both the Behavior Pack and Resource Pack of Admin Suite. Also enable the Beta APIs toggle in the Experiments section. Masuk ke menu Edit Dunia. Pastikan memasang Behavior Pack dan Resource Pack dari Admin Suite. Aktifkan juga opsi Beta APIs di bagian Eksperimen.

4

In-Game VerificationVerifikasi Dalam Game

Enter your world. Use the Admin item or type the /as:info command. The system will prompt you to enter your 16-Digit License Key found in the Licenses tab. You're done! Masuk ke dunia Anda. Gunakan item Admin atau ketik perintah /as:info. Sistem akan meminta Anda memasukkan 16 Digit Kunci Lisensi yang ada di menu Lisensi. Selesai!
