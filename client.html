Installation Guide

1

Download File

Open the My Add-ons tab and click the Download Latest button. Wait for the file to finish downloading to your device.

2

Install to Minecraft

If you downloaded the Realms version, simply double-click it. If you downloaded the BDS version for servers, extract it into your `behavior_packs` and `resource_packs` folders respectively.

3

Apply to World

Go to your World Edit menu. Make sure to apply both the Behavior Pack and Resource Pack of Admin Suite. Also enable the Beta APIs toggle in the Experiments section.

4

In-Game Verification

Enter your world. Use the Admin item or type the /as:info command. The system will prompt you to enter your 16-Digit License Key found in the Licenses tab. You're done!
